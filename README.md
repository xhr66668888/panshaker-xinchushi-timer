# 海外工时记录系统 (Node.js 版)

这是一个为 Panshaker 研发和市场部门设计的轻量级、Win98 风格的工时记录系统。
主要用于统计国内员工耗费在海外项目上的工时，方便进行成本核算。

## 项目结构
- `index.js` - 后端与 API 服务器核心代码 (Node.js + Express)
- `package.json` - 依赖配置表
- `public/` - 前端文件 (包含 `index.html` UI 界面, `stats.html` 核算界面)
- `work_records.db` - (首次运行自动生成) SQLite 数据库文件，用于持久化保存所有的工时数据

## 使用说明与账号权限

本系统加入了简单的权限隔离机制，打开网页后会要求输入密码：
- **员工打卡密码：`8888`** (登录后仅可填报工时，以及撤销自己的错误填报)
- **老板核算密码：`9999`** (登录后仅可查看和导出当月的所有打卡统计)

## 部署教程 (Windows Server)

本系统采用纯 Node.js 编写，部署在 Windows 服务器上极其简单。

### 第一步：安装 Node.js 环境
1. 在 Windows Server 浏览器中打开 [Node.js 官网 (https://nodejs.org/)](https://nodejs.org/)。
2. 下载并安装最新的 **LTS** (长期支持) 版本（例如 20.x 或 22.x）。
3. 安装过程中，一路点击 **Next (下一步)** 即可，确保勾选了 "Add to PATH"（默认已勾选）。

### 第二步：部署代码
1. 将包含了此 `README.md` 的整个 `PanshakerRelease` 文件夹，复制/上传到 Windows 服务器上（例如放在 `C:\Panshaker\PanshakerRelease`）。

### 第三步：安装系统依赖
1. 在 Windows 服务器上，打开文件夹 `C:\Panshaker\PanshakerRelease`。
2. 在文件资源管理器的地址栏中，清空路径，输入 `cmd` 并按回车键，这会在当前文件夹下打开命令提示符黑窗口。
3. 在黑窗口中输入以下命令并回车，等待安装完成（只需执行一次）：
   ```cmd
   npm install
   ```

### 第四步：启动服务
1. 依赖安装完成后，在同一个黑窗口中输入以下命令启动服务：
   ```cmd
   node index.js
   ```
2. 看到控制台输出 `Server running at http://localhost:5000` 就说明服务已经成功启动并在运行了。

### 第五步：局域网访问与防火墙设置
1. **本地访问**：在云服务器内，通过浏览器访问 `http://localhost:5000` 即可看到界面。
2. **局域网/公网访问**：
   - 您的员工需要通过这台服务器的 **实际 IP 地址** 加上 `:5000` 端口来访问（例如：`http://192.168.1.100:5000`）。
   - **⚠️ 防火墙设置**：Windows Server 默认会拦截外部访问。您需要打开 "高级安全 Windows Defender 防火墙"，在 "入站规则" 中新建规则，允许 **TCP 端口 5000** 的所有连接。
   - 如果您使用的是公有云（如阿里云、腾讯云），您还需要在网页控制台的**安全组**中，放行入方向的 `5000` 端口。

---

### (可选进阶) 第六步：利用 PM2 保持服务永远在线 (防崩溃、开机自启)
如果直接用 `node index.js` 启动，一旦您关掉那个黑窗口，网站就会挂掉。
建议使用 `pm2` 让系统在后台永远运行：

1. 在黑窗口中安装 pm2：
   ```cmd
   npm install pm2 -g
   ```
2. 使用 pm2 启动这个应用：
   ```cmd
   pm2 start index.js --name PanshakerWorkRecord
   ```
这样，即使您关闭了命令提示符，网页系统也会一直稳定运行了。
